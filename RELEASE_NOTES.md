# DTS Audio Monitor v1.2.0

## Исправления

- **DTS Sound Unbound больше не открывается** при автоматической проверке каждые N секунд
- Проверка только через **реестр Windows** + тихий **SoundVolumeView**
- DTS-приложение — только при ручной кнопке и только если в `config.json` включено `UseDtsAppOnManualFix: true` (по умолчанию выкл.)
- Исправлен ложный «чужой формат» — не зацикливается на неизвестных GUID
- Авто-fix не чаще **1 раза в 60 секунд**

## Установка

1. Скачайте `DtsAudioMonitor-v1.2.0-win-x64.zip` из Assets.
2. Распакуйте и запустите **`DtsAudioMonitor.exe`**.
