# DTS Audio Monitor v1.1.2

## Что нового

- Безрамочное окно без стандартной шапки Windows
- Свои кнопки «Свернуть» и «Скрыть в трей», перетаскивание за верхнюю панель
- Скруглённые углы, тень, мягкое свечение и обновлённые карточки

## Установка

1. Скачайте `DtsAudioMonitor-v1.1.2-win-x64.zip` из Assets.
2. Распакуйте в любую папку.
3. Запустите **`DtsAudioMonitor.exe`** или **`Start DTS Audio Monitor.bat`**.
4. (Опционально) **`Install autostart.bat`** — автозагрузка Windows.

## Требования

- Windows 10/11 x64
- [DTS Sound Unbound](https://apps.microsoft.com/detail/9pj0nkl8mcsj)
- .NET **не требуется** (self-contained)
