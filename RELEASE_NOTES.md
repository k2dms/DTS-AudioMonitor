# DTS Audio Monitor v1.1.9

## Что нового

- Мониторинг **формата** пространственного звука на наушниках (не только вкл/выкл)
- Авто-восстановление **DTS Headphone:X**, если выбран Windows Sonic, Dolby и т.п.
- Работает и когда активен **монитор** — без переключения устройства (только правка spatial на HyperX)
- После успешного fix сохраняется «золотой» GUID в `spatial-state.json` для вашего ПК

## Установка

1. Скачайте `DtsAudioMonitor-v1.1.9-win-x64.zip` из Assets.
2. Распакуйте и запустите **`Install Start Menu.bat`**.
3. Запустите **`DtsAudioMonitor.exe`**.
