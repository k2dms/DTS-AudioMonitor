# DTS Audio Monitor v1.0.0

Portable сборка для Windows 10/11 x64. **Не требует установки .NET** — всё включено в архив.

## Установка

1. Скачайте `DtsAudioMonitor-v1.0.0-win-x64.zip` из Assets.
2. Распакуйте в любую папку (например `C:\Apps\DtsAudioMonitor`).
3. Запустите **`DtsAudioMonitor.exe`** или **`Start DTS Audio Monitor.bat`**.
4. (Опционально) **`Install autostart.bat`** — добавить в автозагрузку Windows.

## Требования

- Windows 10/11, 64-bit
- [DTS Sound Unbound](https://apps.microsoft.com/detail/9pj0nkl8mcsj) из Microsoft Store (лицензия / пробный период)
- Наушники и монитор как в `config.json` (по умолчанию HyperX Cloud III и XV272U F3)

## Поведение

- **Headphones** — только проверка и включение DTS Headphone:X, устройство не переключается.
- **Переключение на XV272U F3** — автоматический полный цикл DTS и возврат на монитор.

Иконка в системном трее. Двойной щелчок — открыть окно настроек.
